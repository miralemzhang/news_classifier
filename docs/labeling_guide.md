## 8类新闻标签定义

1. 时政 (Politics)
定义：涉及国家治理、政府活动、政策制定与执行、国际关系的新闻报道。

包含内容：

国家领导人活动（出访、会见、讲话、视察）
政府政策发布（法规、条例、白皮书、规划）
外交事务（国际会议、双边/多边关系、领事事务）
政党活动（党建、代表大会、干部任免）
议会/国会活动（立法、听证、预算审议）
选举与政治运动（候选人、投票、竞选）
政府预算与财政政策
停火谈判、和平协议、外交斡旋
政府主导的难民/移民政策
国际组织活动（联合国、欧盟决议等）
政府发布的白皮书、政策报告（新闻报道其政策内容时）
关键词：总统、总理、部长、外交部、政府、政策、法案、条约、峰会、访问、选举、任命、国务院、两会

注意：白皮书/报告的文档介绍页本身（非新闻报道）→ 其他

2. 经济 (Economy)
定义：涉及金融市场、商业活动、企业运营、宏观经济的新闻报道。

包含内容：

股票/债券/外汇/期货市场动态
企业财报、并购、投资、融资
银行利率、货币政策、央行动态
房地产市场（房价、楼市政策）
GDP、CPI、就业数据等宏观指标
国际贸易（进出口、关税、贸易协定）
企业投资落地、产业园区建设
行业分析报告（非政府发布）
消费趋势、零售数据
关键词：股市、基金、利率、财报、房价、GDP、贸易、投资、并购、融资、上市、破产、通胀

3. 军事 (Military)
定义：涉及军事力量、武装冲突、国防建设的新闻报道。

包含内容：

军事演习、军事部署
武器装备（研发、采购、展示）
战争冲突报道（战况、伤亡、前线动态）
空袭、轰炸、交火事件
国防预算与军队建设
军事同盟与军售
退伍军人事务
关键词：军队、士兵、武器、战争、空袭、轰炸、导弹、坦克、航母、演习、国防、前线、伤亡

注意：战争中的外交谈判→时政；人道危机报道→社会

4. 社会 (Society)
定义：涉及民生、公共事务、社会现象的新闻报道。

包含内容：

教育新闻（学校、考试、政策）
医疗健康（医院、疾病、公共卫生）
交通事故、自然灾害
刑事案件、治安事件
公共安全与应急事件
人道主义危机（饥荒、难民生活）
社区活动、节庆展览
社会热点话题
环保、气候变化（非导航页）
劳工权益、社会福利
人口问题（民众生育意愿、养育成本、老龄化现象）
关键词：学校、医院、事故、犯罪、灾害、救援、民生、社区、公益、志愿者

注意：人口政策调整（如一孩→三孩政策）→ 时政；民众生育意愿、养育成本 → 社会

5. 科技 (Technology)
定义：涉及科学研究、技术创新、数字产品的新闻报道。

包含内容：

人工智能（技术突破、产品发布）
互联网/移动互联网新闻
手机/电脑等数码产品
航天航空（火箭发射、卫星、太空探索）
新能源技术（电池、光伏、储能）
芯片/半导体行业
生物技术、基因工程
科研成果发布
关键词：AI、芯片、火箭、卫星、5G、量子、机器人、新能源、半导体、算法

注意：科技公司的商业投资/并购→经济

6. 体育 (Sports)
定义：涉及体育赛事、运动员、体育组织的新闻报道。

包含内容：

足球、篮球等球类运动
奥运会、世界杯、亚运会
NBA、CBA、英超等职业联赛
运动员动态（转会、受伤、退役）
电子竞技（电竞赛事、战队）
体育政策与组织（国际奥委会、FIFA）
关键词：比赛、冠军、进球、联赛、奥运、世界杯、运动员、教练、转会

7. 娱乐 (Entertainment)
定义：涉及影视、音乐、明星、文化娱乐活动的新闻报道。

包含内容：

明星八卦（恋情、婚姻、绯闻）
电影/电视剧（上映、票房、评论）
综艺节目、选秀节目
音乐（演唱会、新专辑、音乐节）
颁奖典礼（奥斯卡、格莱美、金鸡奖）
网红/直播/短视频
游戏发布（非电竞赛事）
关键词：明星、电影、综艺、演唱会、票房、颁奖、网红、直播、粉丝

8. 其他 (Other)
定义：不属于以上7类的内容，或无法明确判断主题的内容。

包含内容：

机构/公司介绍页面（content_type = institution_profile）
网站导航/目录/归档页面（content_type = navigation_or_index）
用户手册、技术文档、FAQ（content_type = howto_or_doc）
隐私政策、用户协议（content_type = legal_or_contact）
广告软文、营销内容
个人随笔、情感故事
生活服务（宠物、留学中介广告等）
彩票开奖结果
旅游指南（非新闻事件）
无法判断主题的内容
